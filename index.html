Crystal Collector

Crystal Collector is an interactive game built with jQuery. At the start of each game, a random target number is displayed in the top-left corner of the screen. Each magic crystal has a hidden numeric value, and players click on the crystals to add to their total score. The goal is to match the target number exactly — if the player reaches the number, they win; if they exceed it, they lose. This project demonstrates dynamic DOM manipulation, event handling, and game logic using jQuery.

First Step

First Step is a three-in-one educational game for children, developed as a group project using React and jQuery. The platform is designed to teach foundational learning skills through interactive games in subjects such as science, the alphabet, spelling, and typing. By combining fun gameplay with educational content, First Step provides an engaging way for children to build essential skills while exploring interactive learning experiences.

Richies Fighter

Richie’s Fighter is a memory-based counting game built with React. Players click on characters, which shuffle randomly after each selection. The goal is to click on all characters without selecting the same one twice. The game tracks your current score, reflecting your wins, and a high score, showing the maximum number of consecutive characters clicked correctly. This project demonstrates dynamic state management, interactive UI design, and React component handling.
